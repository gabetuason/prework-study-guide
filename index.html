HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The p element represents a paragaph or block of text.
  • The br element creates a line of empty space or a line break between two blocks of content
  • The footer element appears at the bottom of the page containing author, copyright contact, sitemap and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Internal CSS contains CSS rules for the web page in head section of the HTML file.
  • External CSS allows developers to keep all their CSS rules in a separate file, making changing much easier.
  • Inline CSS when a developer wants to style an element directly in the HTML file.
  • href specifies the location of the external sources

Git

  • git add -A adds starter code to the current working branch.
  • git commit -m "add comment here" commit message describing what the commit contains.
  • git pull origin main, to check if its up to date pulled from the github repository.
  • git push origin feature/code. Requesting a change, soon to be merged in the main branch.

    • JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • Array Index can store any type of data such as strings.
    • If statements are a way of introducing decision making into the code
    • A for loop uses a predictable pattern of indices to perform a task on all the items in an array
